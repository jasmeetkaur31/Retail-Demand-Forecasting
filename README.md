# Retail Demand Forecasting & Sales Intelligence Platform

End-to-end retail sales forecasting and analytics project built using Python, SQL, XGBoost, SHAP, and Power BI.

## Tech Stack

* Python
* MySQL
* XGBoost
* SHAP
* Power BI
* Pandas

## Results

* R² Score: 0.935
* MAPE: 7.97%

## Key Insights

* Promotions increased average sales by 39.5%
* Customer traffic was the strongest sales driver
* Store Type 1 achieved the highest average sales
